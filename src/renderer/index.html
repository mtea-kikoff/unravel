Unravel

One-time setup

Unravel talks to Gmail directly from this Mac — nothing passes through anyone else's server. That means it needs its own Google API credentials, created once:

  1. Open the and create (or pick) a project.
  2. Enable the and the (Drive powers downloading files people share as links).
  3. Configure the OAuth consent screen (External is fine — add your own address as a test user).
  4. Create an OAuth client ID with application type Desktop app, then paste both values here.

Connect Gmail

Your browser will open so Google can confirm it's you. Unravel only asks for read-only access to Gmail and Drive — it can't send, change, or delete anything.

Searches use Gmail syntax — try from:brunner has:attachment. Empty search shows recent threads with attachments.
